# 📄 AI Smart Document Analyzer

An AI-powered Document Analyzer built with **Python**, **Streamlit**, and **Ollama**.

## 🚀 Features

- 📄 PDF Reader
- 📝 DOCX Reader
- 📃 TXT Reader
- 🤖 AI Summary
- 🔑 Keyword Extraction
- 💬 Ask AI
- 📥 Download Report
- 📊 Document Statistics

---

## 🛠 Technologies

- Python
- Streamlit
- Ollama
- Llama 3.2
- PyPDF2
- python-docx

---

## ▶️ Installation

```bash
pip install -r requirements.txt
```

Run Ollama:

```bash
ollama serve
```

Download model:

```bash
ollama pull llama3.2:1b
```

Run application:

```bash
streamlit run app.py
```
